PROJECT TITLE: Graphing Calculator 


PURPOSE OF PROJECT:

To better understand the functions that go behind the graphing calcualtor


USER INSTRUCTIONS:
Type a function into "y = " box then press graph. Do not put any spaces.

You can set:
xMin, xMax of the window
yMin, yMax of the window

Roots window shows the roots where y=0.

You can specify to what decimal place the roots should round to by specifying it in 
"Round to" box. ex) 2 would mean to the 100th place.

"Change graph color" lets you change the background color of the calculator.
